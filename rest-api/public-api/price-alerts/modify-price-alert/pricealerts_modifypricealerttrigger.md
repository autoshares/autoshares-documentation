# Syntax

## Modify price alert

```text
PUT /v{version}/users/{userId}/pricealerts/{alertId}
```

### Description

Modify price alert. Model properties "State",

| **Header** | **Et-App-Key**   _required_ | Application key | string |  |
| **Path** | **alertId**   _required_ | Alert identifier | integer \(int32\) |  |
